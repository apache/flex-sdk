*** Adobe Flex Component Kit for Flash CS3

This folder contains the source files for the Flex Component Kit for Flash CS3. If you wish to install the Flex Component Kit for Flash CS3 you can find the FlexComponentKit.mxp in the following directory:

<SDK install_dir>/frameworks/flash-integration/


For more information on using the Flex Component Kit for Flash CS3, see "Importing Flash CS3 Assets into Flex" in the Flex Developer's Guide at http://www.adobe.com/go/learn_flex3.
